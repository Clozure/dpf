Picture Window

Simple Instructions

Use the File menu to create a new slideshow window.

You can create as many slideshow windows as you'd like.

Slideshow windows stay on top by default. Use the View menu or Preferences to change that.

You can make an always-on-top window transparent by moving the mouse over it.

Preferences affect what happens to new windows. To change a window that already exists use the View menu.

Picture Window will remember the slideshows you have open and restore them when you quit and restart the app.
